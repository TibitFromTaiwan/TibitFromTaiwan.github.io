高效率系列#1 : Windows系統常用快捷鍵

Tibittw | 週日 30 五月 2021

緣由：自己本身是一個很習慣用快捷鍵~~偷懶~~加快做事速度的人，也希望可以分享一些常用的快捷鍵，加快一些瑣碎步調的速度。

檔案管理

我們先從最常用的資料總管(explorer.exe)、或者通俗點平常說的「資料夾」開始說起吧。

  1. 新增資料夾 : 滑鼠右鍵 + W + F
  2. 重新命名 : 選取後按F2
  3. 將檢視方式改為「詳細資料」模式 : 滑鼠右鍵 之後 V 之後 D
  4. 重新整理 : Ctrl + R 或 F5，常用在複製完資料或下載好檔案，需要讓檔案重新排序時
  5. 在新視窗開啟要開啟的資料夾 : Ctrl + 滑鼠雙擊左鍵
  6. 關閉視窗 : 除了Alt + F4 外，可以用Ctrl + W 比較符合人體工學(?)
  7. 快速完全刪除(不經過資源回收桶) : Alt + Del

視窗系統

  1. 快速回到桌面 : Win + D，或鼠標移到螢幕右下角(沒有改過工作列的位置的話)按一下。
  2. 切換視窗 : Alt + Tab可以直接用鍵盤控制，Win + Tab 需要再用滑鼠選要切換的視窗，不過後者有管理虛擬桌面的功能。
  3. 將視窗快速貼齊螢幕的上下左右 : Win + 上下左右鍵，會讓視窗自動填滿螢幕的右半邊、左下部分等，具體怎麼運作試過就知道😎，在需要將螢幕分成兩邊時非常方便。

其他雜項

  1. 快速開啟工作列的程式 : Win + 1 ~ 0，數字分別對應工作列前十個程式。舉例來說，我的工作列第一項為檔案總管，第二項為Google Chrome， 則Win + 1會開啟檔案總管，Win + 2會開啟Chrome瀏覽器。

  2. 快速關機 : Win + D 之後 Alt + F4 之後Enter

  3. 隱藏桌面資料夾 : 在桌面按右鍵後依序按V, D 我會習慣將桌面的內容隱藏，所以算是小眾用途

以上就是一些Windows系統內我自己常用的快捷鍵，
